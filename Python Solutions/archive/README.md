# Archive - Additional LeetCode Questions

This folder contains template files for popular LeetCode questions that haven't been implemented yet. When you're ready to work on a question, move it to the appropriate category folder and implement the solution.

## How to Use

1. Browse the questions in this folder
2. When you want to work on a question, move it to the appropriate category:
   ```bash
   mv archive/LC0020_ValidParentheses.py stack_queue/
   ```
3. Implement the solution
4. Add test cases
5. Run tests to verify
